# HR Employee Registration System

A simple HR employee registration system built with Python and Flask.

## Requirements

- Python 3 (download from https://www.python.org)

## How to Run

**Step 1 — Install dependencies**
```
pip install -r requirements.txt
```

**Step 2 — Start the app**
```
python hr_system.py
```

**Step 3 — Open browser**
```
http://127.0.0.1:5000
```

## Default Login Accounts

| Role       | Username | Password   | Access              |
|------------|----------|------------|---------------------|
| HR Admin   | admin    | admin123   | Register + View     |
| HR Viewer  | viewer   | viewer123  | View only           |

> The database (`database.db`) is created automatically on first run.
